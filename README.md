## What problems to solve ?
在IOS中，双击会导致页面向上滚动，解决方案是，阻止 `document` 元素的 `dblclick` 事件。

## Install

``` bash
# use npm 
npm i stop-dblclick-scroll
# use yarn
yarn add stop-dblclick-scroll
```

## Usage

``` javascript
// ES6
import 'stop-dblclick-scroll';
// require
require('stop-dblclick-scroll');
```
